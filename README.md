# Advanced_Excel_Project

Hospital Emergency Room Dashboard

Project Overview :
This project focuses on analyzing hospital emergency room (ER) patient data using Advanced Excel to uncover key operational insights. The objective is to understand patient flow, admission trends, wait-time patterns, and department-wise demand to support data-driven decision-making in healthcare operations.

Dataset :
It contains 9,000+ emergency room patient records
Raw data provided in CSV format
It includes:
Patient demographics
Visit details
Wait times
Admission status
Department information

Project Objectives :
Analyze ER admission patterns and peak visit hours
Identify wait-time trends across age groups and departments
Measure key performance indicators (KPIs) for ER efficiency
Build an interactive Excel dashboard for quick and clear insights

Tools & Techniques Used :
Microsoft Excel (Advanced)
Data cleaning and transformation using formulas and calculated fields
Age group classification and categorical analysis
Pivot Tables and Pivot Charts
Slicers for interactive filtering
KPI metrics for performance tracking

Dashboard Features :
Interactive filters for department, age group, and time period
Visualization of patient volume and average wait times
Admission vs non-admission analysis
KPI indicators for emergency room performance

Key Insights & Outcomes :
Identified high-demand departments and peak ER hours
Highlighted patient age groups experiencing longer wait times
Provided insights to support staffing optimization and resource allocation
